# Система мониторинга (Prometheus + Grafana)

Данный проект представляет собой мониторинг системных ресурсов (CPU, RAM, Disk I/O, Network) с использованием стека Prometheus и Grafana. Развертывание осуществляется в контейнерах Docker.

## Описание архитектуры

Проект реализует классическую схему сбора метрик (Pull-модель):

1.  **Node Exporter:** Агент, запускаемый на хосте. Собирает технические метрики операционной системы (Linux) и отдает их в формате HTTP по адресу `/metrics`.
2.  **Prometheus:** Сервер сбора данных (Time Series Database). Периодически опрашивает Node Exporter, сохраняет полученные данные и предоставляет язык запросов (PromQL) для их выборки.
3.  **Grafana:** Платформа для визуализации. Подключается к Prometheus, выполняет запросы и отображает данные в виде графиков и дашбордов.

## Стек технологий

*   **Prometheus:** Сбор и хранение метрик.
*   **Node Exporter:** Экспортер системных показателей Linux.
*   **Grafana:** Визуализация данных.
*   **Docker & Docker Compose:** Контейнеризация и оркестрация сервисов.

## Установка и запуск

Для работы требуется установленный Docker и Docker Compose.

1.  Клонирование репозитория:
    ```bash
    git clone https://github.com/charnes-test/monitoring.git
    cd monitoring-prometheus
    ```

2.  Запуск контейнеров в фоновом режиме:
    ```bash
    docker compose up -d
    ```

3.  Проверка статуса контейнеров:
    ```bash
    docker ps
    ```
    Должны быть активны три контейнера: `prometheus`, `grafana`, `node-exporter`.

## Настройка Grafana

После запуска веб-интерфейс Grafana доступен по адресу: `http://localhost:3000`

**Данные для входа по умолчанию:**
*   **Логин:** `admin`
*   **Пароль:** `admin` (система предложит сменить пароль при первом входе).

### Подключение источника данных (Data Source)

Для отображения графиков необходимо подключить Prometheus к Grafana:

1.  Перейдите в **Configuration** (шестеренка) -> **Data Sources**.
2.  Нажмите **Add data source** и выберите **Prometheus**.
3.  В поле **URL** укажите адрес внутри сети Docker:
    ```text
    http://prometheus:9090
    ```
4.  Нажмите **Save & Test**. Должно появиться сообщение об успешном подключении.

### Импорт Дашборда

Для быстрой настройки визуализации рекомендуется использовать готовый шаблон:

1.  Перейдите в меню **Dashboards** -> **Import**.
2.  В поле "Import via grafana.com" введите ID: **1860** (Node Exporter Full).
3.  Нажмите **Load**.
4.  В выпадающем списке выберите созданный ранее источник данных Prometheus.
5.  Нажмите **Import**.

## Отслеживаемые метрики

После настройки на дашборде будут доступны следующие показатели:
*   Загрузка процессора (CPU Usage) по ядрам.
*   Использование оперативной памяти (RAM) и SWAP.
*   Свободное место на дисках и операции ввода/вывода (I/O).
*   Сетевой трафик (Загрузка/Отдача).
*   Время непрерывной работы системы (Uptime).
